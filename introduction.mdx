# LinkupAPI Documentation

LinkupAPI connects your product or AI agent to LinkedIn, WhatsApp, and real-time B2B intelligence. One unified API to automate outreach, find contacts, and gather market insights for **sales** and **recruitment**.

## Three API Categories

**Engagement API** - Automate LinkedIn and WhatsApp interactions
- LinkedIn: Invitations, Messages, Comments
- WhatsApp: Messaging with media support

**Mail API** - Find and verify professional emails
- Email Finder, Verifier, Reverse Lookup

**Web Crawler API** - Real-time B2B intelligence (public data only)
- People Search, Company Intelligence, Hiring Signals, Funding Data

## Quick Start

### 1. Get Your API Key
Sign up at [linkupapi.com](https://www.linkupapi.com)

### 2. Make Your First Request
```bash
curl -H "x-api-key: your_api_key" \
  https://api.linkupapi.com/v1/people/search
```

### 3. Explore the Docs
Browse endpoints and start building your automation.

## Authentication

Include your API key in the `x-api-key` header:
```bash
curl -H "x-api-key: your_api_key" \
  https://api.linkupapi.com/v1/...
```
