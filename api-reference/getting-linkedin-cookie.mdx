# Getting LinkedIn Authentication Cookie

To use most endpoints of the LinkUp API, you need a valid LinkedIn authentication cookie (li_at). There are two methods to obtain this cookie, each with its own considerations.

## Method 1: Manual Extraction

This method involves manually logging into LinkedIn and extracting the cookie. While simpler, it requires careful attention to IP addresses and locations.

### Steps:

1. **Install Cookie Editor Extension**
   - For Chrome: Install [EditThisCookie](https://chrome.google.com/webstore/detail/editthiscookie/fngmhnnpilhplaeedifhccceomclgfbg)
   - For Firefox: Install [Cookie Editor](https://addons.mozilla.org/en-US/firefox/addon/edit-cookie/)

2. **Login to LinkedIn**
   - Go to [LinkedIn.com](https://www.linkedin.com)
   - Login with your account credentials
   - Make sure you complete any security verifications if prompted

3. **Extract the Cookie**
   - Click on the EditThisCookie extension icon in your browser
   - Look for the cookie named "li_at"
   - Copy its value - it should be a long string of characters

### ⚠️ Important Considerations

When using this method, be aware of these critical points:

1. **IP Address Matching**
   - The IP address/location you use to log in MUST match the one you'll use with our API
   - Using different IPs can trigger LinkedIn's security measures
   - This can lead to cookie invalidation or account security checks

2. **Cookie Lifespan**
   - Manually extracted cookies typically last 90 days
   - However, they can expire earlier if:
     - LinkedIn detects suspicious activity
     - You change your password
     - You log out of LinkedIn

3. **Security Risks**
   - Never share your li_at cookie
   - It provides full access to your LinkedIn account
   - Store it securely like any other authentication token

## Method 2: Using LinkUp Authentication API (Recommended)

Our authentication API provides a more reliable and secure way to obtain the LinkedIn cookie. This method automatically handles IP matching and security challenges.

### Advantages:

1. **Automatic IP Handling**
   - Cookie is automatically configured to work with our proxy network
   - No need to worry about IP matching

2. **Security Management**
   - Handles LinkedIn security challenges automatically
   - Manages verification codes when required
   - Reduces risk of account restrictions

3. **Better Reliability**
   - Properly formatted cookies every time
   - Automatic handling of security updates
   - Compatible with all API endpoints

### How to Use:

1. **Login Request**
```bash
curl -X POST https://api.linkupapi.com/v1/auth/login \
  -H "x-api-key: your_api_key" \
  -d "email=your_linkedin_email" \
  -d "password=your_linkedin_password"
```

2. **Handle Verification (if required)**
```bash
curl -X POST https://api.linkupapi.com/v1/auth/verify \
  -H "x-api-key: your_api_key" \
  -d "code=verification_code" \
  -d "email=your_linkedin_email"
```

### Response Example:

```json
{
  "status": "success",
  "message": "Login successful",
  "li_at": "XXXX..."  // Your LinkedIn cookie
}
```

## Best Practices

1. **Cookie Storage**
   - Store the cookie securely
   - Treat it like a sensitive authentication token
   - Never expose it in client-side code

2. **Error Handling**
   - Implement cookie refresh logic
   - Watch for 403 errors indicating cookie expiration
   - Have a backup authentication flow ready

3. **Cookie Rotation**
   - Plan for regular cookie rotation
   - Monitor cookie expiration
   - Implement automatic re-authentication when needed

4. **Security**
   - Use environment variables for cookie storage
   - Never commit cookies to version control
   - Implement proper access controls

## Troubleshooting

Common issues and solutions:

1. **Cookie Invalid**
   - Check if using correct IP/location
   - Verify cookie format is correct
   - Try re-authenticating

2. **Frequent Invalidation**
   - Reduce request frequency
   - Check for IP mismatches
   - Use our authentication API instead

3. **Security Challenges**
   - Switch to our authentication API
   - Reduce automation frequency
   - Ensure account is in good standing
