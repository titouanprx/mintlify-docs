title: 'Search Companies'
api: 'POST /v1/linkedin/companies/search'
description: 'Search for companies on LinkedIn'
Search for companies using various filters and criteria.
Header Parameters
<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>
Body Parameters
<ParamField body="login_cookie" type="string" required>
  LinkedIn authentication cookie
</ParamField>
<ParamField body="keyword" type="string">
  Search keyword
</ParamField>
<ParamField body="location" type="string">
  Company location
</ParamField>
<ParamField body="sector" type="string">
  Industry sector
</ParamField>
<ParamField body="company_size" type="string">
  Size of the company
</ParamField>
<ParamField body="total_results" type="integer" default="10">
  Number of results to return
</ParamField>
<RequestExample>
```bash
curl -X POST 'https://api.linkupapi.com/v1/linkedin/companies/search' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'keyword=technology' \
-d 'location=London' \
-d 'sector=Software' \
-d 'login_cookie=your-cookie'
```
</RequestExample>
<ResponseExample>
```json
{
  "status": "success",
  "data": {
    "total": 125,
    "companies": [
      {
        "name": "Tech Solutions Ltd",
        "url": "https://linkedin.com/company/tech-solutions",
        "industry": "Software",
        "location": "London, UK",
        "size": "51-200 employees"
      }
    ]
  }
}
```
</ResponseExample>
