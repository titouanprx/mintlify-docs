---
title: 'Search Companies'
api: 'POST /v1/linkup/companies/search'
description: 'Rechercher des entreprises sur LinkedIn'
---

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Votre clé API
</ParamField>

### Body Parameters

<ParamField body="location" type="string">
  Localisation de l'entreprise
</ParamField>

<ParamField body="sector" type="string">
  Secteur d'activité
</ParamField>

<ParamField body="keyword" type="string">
  Mot-clé pour la recherche
</ParamField>

<ParamField body="company_size" type="string">
  Taille de l'entreprise
</ParamField>

<ParamField body="total_results" type="integer" default="5">
  Nombre de résultats à retourner
</ParamField>

<ParamField body="login_cookie" type="string" required>
  Cookie de session LinkedIn
</ParamField>

### Response

<ResponseField name="companies" type="array">
  Liste des entreprises trouvées
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.example.com/v1/linkup/companies/search' \
-H 'x-api-key: votre-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'keyword=tech' \
-d 'location=Paris' \
-d 'login_cookie=votre-cookie'
```
</RequestExample>

<ResponseExample>
```json
{
  "companies": [
    {
      "name": "TechCorp",
      "industry": "Technology",
      "size": "1001-5000",
      "location": "Paris, France",
      "company_url": "https://linkedin.com/company/techcorp"
    }
  ]
}
```
</ResponseExample>
