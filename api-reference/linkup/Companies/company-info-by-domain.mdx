title: 'Get Company Information by Domain'
api: 'POST /v1/data/company/info-by-domain'
description: 'Search and extract company information from LinkedIn using a domain name'

Search for a company on LinkedIn using its domain name and extract comprehensive information including description, size, locations, and other business details.
<Note>
  This endpoint costs **2 credits** per request (1 credit for Google search + 1 credit for LinkedIn scraping).
</Note>

### How it works

1. **Domain Search**: The endpoint first searches Google to find the LinkedIn company page associated with the provided domain
2. **URL Extraction**: It extracts the LinkedIn company URL from the search results
3. **Data Scraping**: Using the found LinkedIn URL, it scrapes comprehensive company information with automatic proxy rotation
4. **Data Return**: Returns the complete company profile data along with the LinkedIn URL and searched domain

### Use Cases

- **Lead Generation**: Find company information when you only have a domain name
- **Competitor Research**: Gather detailed information about competitors using their website domains
- **Data Enrichment**: Enrich your database with LinkedIn company data using domain names
- **Market Research**: Research companies in your target market using their domain names

### Rate Limits

This endpoint uses 2 credits per request:
- 1 credit for the Google search to find the LinkedIn URL
- 1 credit for the LinkedIn company data scraping

### Error Handling

The endpoint handles various error scenarios:
- **401**: Insufficient credits
- **400**: Missing or invalid domain parameter
- **404**: No LinkedIn company page found for the domain
- **500**: Service temporarily unavailable or scraping errors
