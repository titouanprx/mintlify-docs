---
title: 'Extract Posts'
api: 'POST /v1/linkup/posts/extract'
description: 'Extract LinkedIn posts'
---

Extract posts from LinkedIn profiles or feeds.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="post_type" type="string">
  Type of posts to extract
</ParamField>

<ParamField body="sort_by" type="string" default="date_posted">
  Sort criteria for the posts
</ParamField>

<ParamField body="keyword" type="string">
  Keyword to filter posts
</ParamField>

<ParamField body="post_date" type="string">
  Filter posts by date
</ParamField>

<ParamField body="linkedin_url" type="string">
  LinkedIn URL to extract posts from
</ParamField>

<ParamField body="total_results" type="integer" default="10">
  Number of posts to return
</ParamField>

<ParamField body="login_cookie" type="string" required>
  LinkedIn session cookie
</ParamField>

### Response

<ResponseField name="posts" type="array">
  List of extracted posts
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.example.com/v1/linkup/posts/extract' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'login_cookie=your-cookie' \
-d 'total_results=10'
```
</RequestExample>

<ResponseExample>
```json
{
  "posts": [
    {
      "id": "post123",
      "content": "Post content",
      "date": "2024-01-01",
      "likes": 42,
      "comments": 5
    }
  ]
}
```
</ResponseExample>
