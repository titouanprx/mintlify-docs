---
title: 'Extract Contact'
api: 'POST /v1/linkup/profile/extract-contact'
description: 'Extraire les informations de contact'
---

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Votre clé API
</ParamField>

### Body Parameters

<ParamField body="linkedin_url" type="string" required>
  URL du profil LinkedIn
</ParamField>

<ParamField body="login_token" type="string" required>
  Token de session LinkedIn
</ParamField>

### Response

<ResponseField name="name" type="string">
  Nom complet du contact
</ResponseField>

<ResponseField name="email" type="string">
  Email du contact
</ResponseField>

<ResponseField name="position" type="string">
  Poste actuel
</ResponseField>

<ResponseField name="company" type="string">
  Entreprise actuelle
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.example.com/v1/linkup/profile/extract-contact' \
-H 'x-api-key: votre-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'linkedin_url=https://linkedin.com/in/username' \
-d 'login_token=votre-token'
```
</RequestExample>

<ResponseExample>
```json
{
  "name": "John Doe",
  "email": "john@example.com",
  "position": "Software Engineer",
  "company": "Tech Corp"
}
```
</ResponseExample>
