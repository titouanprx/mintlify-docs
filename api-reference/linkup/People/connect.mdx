---
title: 'Connect to Profile'
api: 'POST /v1/linkedin/network/connect'
description: 'Send a connection request to a LinkedIn profile'
---

Send a connection request to a LinkedIn user with an optional message.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="linkedin_url" type="string" required>
  URL of the LinkedIn profile you want to connect with
</ParamField>

<ParamField body="message" type="string">
  Optional connection message
</ParamField>

<ParamField body="login_cookie" type="string" required>
  LinkedIn authentication cookie
</ParamField>

<RequestExample>
```bash
curl -X POST 'https://api.linkupapi.com/v1/linkedin/network/connect' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'linkedin_url=https://linkedin.com/in/username' \
-d 'message=Hi, I would like to connect with you' \
-d 'login_cookie=your-cookie'
```
</RequestExample>

<ResponseExample>
```json
{
  "status": "success",
  "message": "Connection request sent successfully"
}
```
</ResponseExample>
