---
title: 'Connect To Profile'
api: 'POST /v1/linkup/network/connect'
description: 'Envoyer une invitation de connexion'
---

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Votre clé API
</ParamField>

### Body Parameters

<ParamField body="linkedin_url" type="string" required>
  URL du profil à contacter
</ParamField>

<ParamField body="message" type="string">
  Message personnalisé de connexion
</ParamField>

<ParamField body="login_cookie" type="string" required>
  Cookie de session LinkedIn
</ParamField>

### Response

<ResponseField name="status" type="string">
  Statut de l'invitation
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.example.com/v1/linkup/network/connect' \
-H 'x-api-key: votre-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'linkedin_url=https://linkedin.com/in/contact' \
-d 'message=Bonjour, je souhaite rejoindre votre réseau' \
-d 'login_cookie=votre-cookie'
```
</RequestExample>

<ResponseExample>
```json
{
  "status": "success",
  "invitation_sent": true
}
```
</ResponseExample>
