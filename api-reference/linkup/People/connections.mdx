---
title: 'Get Connections'
api: 'POST /v1/linkedin/network/connections'
description: 'Retrieve your LinkedIn connections'
---

Get a list of your LinkedIn connections with their basic information.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="login_cookie" type="string" required>
  LinkedIn authentication cookie
</ParamField>

<ParamField body="total_results" type="integer" default="10">
  Number of connections to retrieve
</ParamField>

<RequestExample>
```bash
curl -X POST 'https://api.linkupapi.com/v1/linkedin/network/connections' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'login_cookie=your-cookie' \
-d 'total_results=10'
```
</RequestExample>

<ResponseExample>
```json
{
  "status": "success",
  "data": {
    "total": 500,
    "connections": [
      {
        "firstName": "John",
        "lastName": "Doe",
        "headline": "Software Engineer",
        "profileUrl": "https://linkedin.com/in/johndoe",
        "connectionDegree": "1st"
      }
    ]
  }
}
```
</ResponseExample>
