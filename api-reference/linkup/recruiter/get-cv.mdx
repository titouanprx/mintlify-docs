---
title: 'Get CV'
api: 'POST /v1/recruiter/cv'
description: 'Retrieve a candidate\'s CV from a LinkedIn job application'
---

Retrieve the CV/resume URL for a specific candidate who has applied to your LinkedIn job posting.

<Note>
  This endpoint costs **1 credit** per request.
</Note>

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="application_id" type="string" required>
  LinkedIn application ID of the candidate
</ParamField>

<ParamField body="login_cookie" type="string" required>
  LinkedIn authentication cookie (li_at) obtained from the login/verify process
</ParamField>

<ParamField body="country" type="string" default="FR">
  Country code for proxy selection. Available: (US,UK,FR)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Request status (success/error)
</ResponseField>

<ResponseField name="data" type="object">
  CV data
</ResponseField>

<ResponseField name="data.cv_url" type="string">
  Direct URL to download the candidate's CV
</ResponseField>

<ResponseField name="data.file_name" type="string">
  Original filename of the CV
</ResponseField>

<ResponseField name="data.file_type" type="string">
  File type of the CV (PDF, DOCX, etc.)
</ResponseField>

<ResponseExample>

{
  "status": "success",
