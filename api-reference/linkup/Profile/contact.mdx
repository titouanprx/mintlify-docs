title: 'Extract Contact'
api: 'POST /v1/linkedin/profile/extract-contact'
description: 'Extract contact information from a LinkedIn profile'
Get contact details from a LinkedIn profile when available.
Header Parameters
<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>
Body Parameters
<ParamField body="linkedin_url" type="string" required>
  URL of the LinkedIn profile
</ParamField>
<ParamField body="login_cookie" type="string" required>
  LinkedIn authentication cookie
</ParamField>
<RequestExample>
```bash
curl -X POST 'https://api.linkupapi.com/v1/linkedin/profile/extract-contact' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'linkedin_url=https://linkedin.com/in/username' \
-d 'login_cookie=your-cookie'
```
</RequestExample>
<ResponseExample>
```json
{
  "status": "success",
  "data": {
    "email": "contact@example.com",
    "phone": "+1234567890",
    "twitter": "@username",
    "websites": ["https://example.com"]
  }
}
```
</ResponseExample>
