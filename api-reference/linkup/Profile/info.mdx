title: 'Extract Profile'
api: 'POST /v1/linkedin/profile/extract-info'
description: 'Extract information from a LinkedIn profile URL'
Extract detailed information from any LinkedIn profile using their profile URL.
Header Parameters
<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>
Body Parameters
<ParamField body="linkedin_url" type="string" required>
  URL of the LinkedIn profile to analyze
</ParamField>
<ParamField body="login_cookie" type="string" required>
  LinkedIn authentication cookie
</ParamField>
<RequestExample>
```bash
curl -X POST 'https://api.linkupapi.com/v1/linkedin/profile/extract-info' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'linkedin_url=https://linkedin.com/in/username' \
-d 'login_cookie=your-cookie'
```
</RequestExample>
<ResponseExample>
```json
{
  "status": "success",
  "data": {
    "firstName": "Jane",
    "lastName": "Smith",
    "headline": "Product Manager",
    "location": "London, UK",
    "currentCompany": "Tech Corp",
    "education": [
      {
        "school": "University of London",
        "degree": "Bachelor's Degree",
        "field": "Computer Science"
      }
    ]
  }
}
```
</ResponseExample>
