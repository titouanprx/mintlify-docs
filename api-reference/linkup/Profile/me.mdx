---
title: 'Get My Profile'
api: 'GET /v1/linkedin/profile/me'
description: 'Retrieve current user LinkedIn profile information'
---

This endpoint retrieves the complete profile information for the authenticated LinkedIn user. It provides detailed information about your own LinkedIn profile, including professional experience, education, skills, and other profile sections.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

<ParamField header="linkedin-cookie" type="string" required>
  LinkedIn authentication cookie obtained from login/verify process
</ParamField>

### Response

<ResponseField name="profile" type="object">
  Object containing all profile information
</ResponseField>

<ResponseField name="profile.basic_info" type="object">
  Basic profile information
</ResponseField>

<ResponseField name="profile.experience" type="array">
  Array of professional experiences
</ResponseField>

<ResponseField name="profile.education" type="array">
  Array of educational background
</ResponseField>

<ResponseField name="profile.skills" type="array">
  Array of professional skills
</ResponseField>

<ResponseField name="profile.certifications" type="array">
  Array of professional certifications
</ResponseField>

### Error Responses

- `401`: Invalid or expired authentication cookie
- `403`: Invalid API key
- `404`: Profile not found
- `500`: Server error

<RequestExample>
```bash
curl -X GET 'https://api.linkup.com/v1/linkedin/profile/me' \
-H 'x-api-key: your-api-key' \
-H 'linkedin-cookie: li_at=XXXXXX'
```
</RequestExample>
