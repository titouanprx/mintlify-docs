---
title: 'Login'
api: 'POST /v1/linkup/auth/login'
description: 'Authenticate with LinkedIn credentials'
---

Authenticate with your LinkedIn account. If verification is required, a security code will be sent to your email.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="email" type="string" required>
  Your LinkedIn email address
</ParamField>

<ParamField body="password" type="string" required>
  Your LinkedIn password
</ParamField>

### Response

<ResponseField name="status" type="string">
  Authentication status
</ResponseField>

<ResponseField name="requires_verification" type="boolean">
  Whether security code verification is needed
</ResponseField>

<ResponseField name="cookie" type="string">
  LinkedIn authentication cookie (if no verification required)
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.linkup.com/v1/linkup/auth/login' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'email=user@example.com' \
-d 'password=your-password'
```
</RequestExample>

<ResponseExample>
```json Response with Verification Required
{
  "status": "pending",
  "requires_verification": true,
  "email": "u***@example.com"
}
```

```json Response without Verification
{
  "status": "success",
  "requires_verification": false,
  "cookie": "li_at=XXXXXX"
}
```
</ResponseExample>
