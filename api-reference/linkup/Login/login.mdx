---
title: 'LinkedIn Login'
api: 'POST /v1/linkup/auth/login'
description: 'Authenticate with LinkedIn credentials'
---

Connect to LinkedIn using email and password.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="email" type="string" required>
  LinkedIn account email
</ParamField>

<ParamField body="password" type="string" required>
  LinkedIn account password
</ParamField>

### Response

<ResponseField name="status" type="string">
  Authentication status
</ResponseField>

<ResponseField name="session" type="string">
  LinkedIn session token
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.linkup.com/v1/linkup/auth/login' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'email=user@example.com' \
-d 'password=yourpassword'
```
</RequestExample>

<ResponseExample>
```json
{
  "status": "success",
  "session": "your-linkedin-session"
}
```
</ResponseExample>
