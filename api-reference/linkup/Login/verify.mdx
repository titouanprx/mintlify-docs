---
title: 'Verify Code'
api: 'POST /v1/linkup/auth/verify'
description: 'Verify security code for LinkedIn authentication'
---

Verify the security code sent to your email during the LinkedIn authentication process. This endpoint should be called after receiving the verification code from the login endpoint.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="code" type="string" required>
  The verification code received via email
</ParamField>

<ParamField body="email" type="string" required>
  The email address used for LinkedIn login
</ParamField>

### Response

<ResponseField name="status" type="string">
  Verification status
</ResponseField>

<ResponseField name="cookie" type="string">
  LinkedIn authentication cookie (on successful verification)
</ResponseField>

<ResponseExample>
```json Success Response
{
  "status": "success",
  "cookie": "li_at=XXXXXX"
}
```

```json Error Response
{
  "status": "error",
  "message": "Invalid verification code"
}
```
</ResponseExample>
