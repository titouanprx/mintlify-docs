---
title: 'Verify Security Code'
api: 'POST /v1/linkup/auth/verify'
description: 'Verify LinkedIn security code'
---

Verify the security code received after login attempt.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="code" type="string" required>
  Security code received
</ParamField>

<ParamField body="email" type="string" required>
  Email used for login
</ParamField>

### Response

<ResponseField name="status" type="string">
  Verification status
</ResponseField>

<RequestExample>
```bash
curl -X POST 'https://api.linkup.com/v1/linkup/auth/verify' \
-H 'x-api-key: your-api-key' \
-H 'Content-Type: application/x-www-form-urlencoded' \
-d 'code=123456' \
-d 'email=user@example.com'
```
</RequestExample>

<ResponseExample>
```json
{
  "status": "success",
  "verified": true
}
```
</ResponseExample>
