---
title: "Search Companies"
api: "POST /data/search/companies"
description: "Search LinkedIn company pages with advanced filters"
---

## Description

This endpoint allows you to search LinkedIn company pages using various advanced filtering criteria. It uses Google Search to identify LinkedIn company profiles matching the specified criteria.

## Headers

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

## Body Parameters

<ParamField body="keyword" type="string" required>
  Keyword to search for (company name, industry, etc.)
  
  Can also be provided as `company_name` for backward compatibility
</ParamField>

<ParamField body="industry" type="string">
  Industry sector (e.g., "technology", "finance", "healthcare")
</ParamField>

<ParamField body="location" type="string">
  Location (e.g., "Paris", "France", "New York")
</ParamField>

<ParamField body="company_size" type="string">
  Company size category (e.g., "startup", "small", "medium", "large", "enterprise")
</ParamField>

<ParamField body="employee_range" type="string">
  Employee count range (e.g., "1-50", "51-200", "201-1000", "1000+")
</ParamField>

<ParamField body="funding_stage" type="string">
  Funding stage (e.g., "seed", "series_a", "series_b", "series_c", "ipo")
</ParamField>

<ParamField body="founding_year" type="integer">
  Founding year (e.g., 2020)
  
  - Minimum: 1800
  - Maximum: 2024
</ParamField>

<ParamField body="founding_period" type="string">
  Founding period category (e.g., "startup", "recent", "established", "legacy")
</ParamField>

<ParamField body="num_results" type="integer" default="10">
  Number of results to return
  
  - Minimum: 1
  - Maximum: 50,000
  - Default: 10
</ParamField>

<ParamField body="country" type="string" default="FR">
  Country code for proxy (e.g., "FR", "US", "GB")
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status ("success" or "error")
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Response Data">
    <ResponseField name="results_returned" type="integer">
      Number of results returned in this response
    </ResponseField>
    
    <ResponseField name="companies" type="array">
      List of found companies
      <Expandable title="Company Structure">
        <ResponseField name="url" type="string">
          Company LinkedIn profile URL
        </ResponseField>
        <ResponseField name="title" type="string">
          Company title/name from LinkedIn
        </ResponseField>
        <ResponseField name="description" type="string">
          Company description
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Error message (if status = "error")
</ResponseField>

## Error Codes

<ResponseExample>
```json
{
  "status": "error",
  "message": "keyword (or company_name) is required"
}
```
</ResponseExample>

<ResponseExample>
```json
{
  "status": "error",
  "message": "founding_year must be a valid year between 1800 and 2024"
}
```
</ResponseExample>

- **400**: Invalid or missing parameters
- **401**: Invalid API key or insufficient credits
- **500**: Server error during request processing

## Important Notes

- Each successful search consumes 1 credit
- Credits are only deducted on successful searches
- The endpoint uses an automatic retry system for proxy errors
- The `company_name` parameter is supported for backward compatibility with `keyword`
- All filter parameters are optional - you can combine multiple filters for more precise results
- The `founding_year` parameter must be between 1800 and 2024

## Usage Examples

### Basic search with keyword only
```json
{
  "keyword": "artificial intelligence",
  "num_results": 10
}
```

### Search with industry and location filters
```json
{
  "keyword": "software",
  "industry": "technology",
  "location": "San Francisco",
  "num_results": 25
}
```

### Search with company size and funding filters
```json
{
  "keyword": "fintech",
  "company_size": "startup",
  "employee_range": "1-50",
  "funding_stage": "seed",
  "founding_period": "startup",
  "num_results": 15
}
```
