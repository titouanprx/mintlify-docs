---
title: 'Update Webhook Account'
api: 'PUT /webhooks/accounts/{account_id}'
description: 'Update an existing LinkedIn webhook account'
---

Update the configuration of an existing LinkedIn webhook account.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Path Parameters

<ParamField path="account_id" type="string" required>
  The ID of the account to update
</ParamField>

### Body Parameters

<ParamField body="login_token" type="string">
  New LinkedIn authentication token
</ParamField>

<ParamField body="webhook_url" type="string">
  New webhook URL for notifications
</ParamField>

<ParamField body="country" type="string">
  New country code for proxy selection. Available: (US, UK, FR)
</ParamField>

<ParamField body="account_name" type="string">
  New account name
</ParamField>

### Response

<ResponseField name="id" type="string">
  Account identifier
</ResponseField>

<ResponseField name="login_token" type="string">
  Updated LinkedIn authentication token
</ResponseField>

<ResponseField name="country" type="string">
  Updated country code
</ResponseField>

<ResponseField name="webhook_url" type="string">
  Updated webhook URL
</ResponseField>

<ResponseField name="account_name" type="string">
  Updated account name
</ResponseField>

<ResponseField name="is_active" type="boolean">
  Current monitoring status
</ResponseField>

<ResponseField name="created_at" type="string">
  Account creation timestamp
</ResponseField>

<ResponseField name="updated_at" type="string">
  Last update timestamp
</ResponseField>

### Notes
- All body parameters are optional - only provide the fields you want to update
- If monitoring is active, changes will take effect immediately
- Updating the login token may require restarting monitoring for the changes to take effect
