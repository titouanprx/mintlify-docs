---
title: 'List Webhook Accounts'
api: 'GET /webhooks/accounts'
description: 'Retrieve all registered LinkedIn webhook accounts'
---

Get a list of all LinkedIn accounts registered for webhook monitoring under your API key.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Response

<ResponseField name="accounts" type="array">
  <Expandable title="Account Object">
    <ResponseField name="id" type="string">
      Unique identifier for the account
    </ResponseField>

    <ResponseField name="login_token" type="string">
      LinkedIn authentication token
    </ResponseField>

    <ResponseField name="country" type="string">
      Country code for proxy selection
    </ResponseField>

    <ResponseField name="webhook_url" type="string">
      URL where webhooks are sent
    </ResponseField>

    <ResponseField name="account_name" type="string">
      Account name if provided
    </ResponseField>

    <ResponseField name="is_active" type="boolean">
      Whether monitoring is currently active
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Account creation timestamp
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer">
  Total number of accounts
</ResponseField>

### Notes
- Returns all accounts associated with your API key
- Use the `is_active` field to identify which accounts are currently being monitored
- Account tokens are included in the response for reference
