---
title: 'List Webhook Accounts'
api: 'GET /v1/webhooks/accounts'
description: 'Retrieve all registered webhook accounts'
---

Get a list of all LinkedIn and WhatsApp accounts registered for webhook monitoring under your API key.

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Response

<ResponseField name="accounts" type="array">
  <Expandable title="Account Object">
    <ResponseField name="id" type="string">
      Unique identifier for the account
    </ResponseField>

    <ResponseField name="plateforme" type="string">
      Platform type (linkedin or whatsapp)
    </ResponseField>

    <ResponseField name="account_name" type="string">
      Account name
    </ResponseField>

    <ResponseField name="webhook_url" type="string">
      URL where webhooks are sent
    </ResponseField>

    <ResponseField name="is_active" type="boolean">
      Whether monitoring is currently active
    </ResponseField>

    <ResponseField name="login_token" type="string">
      LinkedIn authentication token (LinkedIn only)
    </ResponseField>

    <ResponseField name="country" type="string">
      Country code for proxy selection (LinkedIn only)
    </ResponseField>

    <ResponseField name="phone_number" type="string">
      Phone number (WhatsApp only)
    </ResponseField>

    <ResponseField name="is_authenticated" type="boolean">
      Whether account is authenticated (WhatsApp only)
    </ResponseField>

    <ResponseField name="needs_qr" type="boolean">
      Whether QR authentication is needed (WhatsApp only)
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Account creation timestamp
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="total" type="integer">
  Total number of accounts
</ResponseField>

### Notes
- Returns all accounts associated with your API key
- Use the `is_active` field to identify which accounts are currently being monitored
- Account tokens are included in the response for reference
