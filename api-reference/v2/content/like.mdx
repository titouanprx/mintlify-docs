---
title: 'Like Post'
api: 'POST /v2/content'
description: 'Like a post.'
---

## Overview

Like a specific post from the authenticated account.

<Note>This endpoint consumes **1 credit** per request.</Note>

## Request

<ParamField body="account_id" type="string" required>
  The unique identifier of the account to use.
</ParamField>

<ParamField body="action" type="string" required>
  Must be `"like"`.
</ParamField>

<ParamField body="params" type="object" required>
  <Expandable title="params properties">
    <ParamField body="post_url" type="string" required>
      The URL of the post to like.
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```json
{
  "account_id": "your-account-id",
  "action": "like",
  "params": {
    "post_url": "https://www.linkedin.com/feed/update/urn:li:activity:7654321098765432100"
  }
}
```
</RequestExample>

## Response

<ResponseField name="success" type="boolean">
  Indicates whether the request was successful.
</ResponseField>

<ResponseField name="data" type="object">
  Confirmation of the like action.
</ResponseField>

<ResponseField name="metadata" type="object">
  <Expandable title="metadata properties">
    <ResponseField name="action" type="string">
      The action that was performed (`"like"`).
    </ResponseField>
    <ResponseField name="account_id" type="string">
      The account ID used for the request.
    </ResponseField>
    <ResponseField name="credits_consumed" type="integer">
      Number of credits consumed (1).
    </ResponseField>
    <ResponseField name="timestamp" type="string">
      ISO 8601 timestamp of the response.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>
```json 200
{
  "success": true,
  "data": {},
  "metadata": {
    "action": "like",
    "account_id": "your-account-id",
    "credits_consumed": 1,
    "timestamp": "2026-03-24T12:00:00Z"
  }
}
```

```json 400
{
  "success": false,
  "error": {
    "code": "INVALID_REQUEST",
    "message": "post_url is required."
  },
  "metadata": {
    "action": "like",
    "account_id": "your-account-id",
    "credits_consumed": 0,
    "timestamp": "2026-03-24T12:00:00Z"
  }
}
```
</ResponseExample>
