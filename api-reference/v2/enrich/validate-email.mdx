---
title: 'Validate Email'
api: 'POST /v2/enrich'
description: 'Verify whether an email address is valid and deliverable.'
---

## Overview

Checks whether an email address is valid and deliverable. Returns syntax validity, deliverability, catch-all status, and a reachability verdict. No LinkedIn account is required for this action.

<Note>This endpoint consumes **1 credit** per request.</Note>

## Request

<ParamField body="action" type="string" required>
  Must be `"validate_email"`.
</ParamField>

<ParamField body="params" type="object" required>
  <Expandable title="params properties">
    <ParamField body="email" type="string" required>
      The email address to validate.
    </ParamField>
  </Expandable>
</ParamField>

<RequestExample>
```json
{
  "action": "validate_email",
  "params": {
    "email": "jane.doe@acme-corp.com"
  }
}
```
</RequestExample>

## Response

<ResponseField name="success" type="boolean">
  Indicates whether the request was successful.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data properties">
    <ResponseField name="email" type="string">The email that was validated (echo of input).</ResponseField>
    <ResponseField name="is_valid" type="boolean">`true` when the email passes syntax + MX + deliverability checks.</ResponseField>
    <ResponseField name="is_catch_all" type="boolean">`true` if the domain accepts all addresses (deliverability is not a guarantee).</ResponseField>
    <ResponseField name="reachability" type="string">SMTP probe verdict: `"safe"`, `"risky"`, or `"unknown"`.</ResponseField>
    <ResponseField name="is_deliverable" type="boolean">`true` if the SMTP server accepts mail for this address.</ResponseField>
    <ResponseField name="syntax_valid" type="boolean">`true` if the address has valid RFC syntax.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="metadata" type="object">
  <Expandable title="metadata properties">
    <ResponseField name="action" type="string">The action that was performed (`"validate_email"`).</ResponseField>
    <ResponseField name="credits_consumed" type="integer">Number of credits consumed (1).</ResponseField>
    <ResponseField name="timestamp" type="string">ISO 8601 timestamp of the response.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseExample>
```json 200
{
  "success": true,
  "data": {
    "email": "jane.doe@acme-corp.com",
    "is_valid": true,
    "is_catch_all": true,

    "syntax_valid": true

  }

```
</ResponseExample>
