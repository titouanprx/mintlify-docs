---
title: 'Withdraw Invitation'
api: 'POST /v2/network'
description: 'Withdraw a pending sent connection invitation.'
---

## Overview

Withdraws a previously sent connection invitation that is still pending. The invitation is identified by its `invitation_id`, which can be obtained from the [List Sent Invitations](/api-reference/v2/network/list-sent) endpoint or from [Check Invitation Status](/api-reference/v2/network/check-invitation).

<Note>This endpoint consumes **1 credit** per request.</Note>

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication.
</ParamField>

<ParamField body="account_id" type="string" required>
  The unique identifier of the account to use.
</ParamField>

<ParamField body="action" type="string" required>
  Must be `"withdraw"`.
</ParamField>

<ParamField body="params" type="object" required>
  <Expandable title="params properties">
    <ParamField body="invitation_id" type="string" required>
      The numeric ID of the invitation to withdraw (e.g. `"7453486530700017665"`). Accepts string or integer.
    </ParamField>
  </Expandable>
</ParamField>


## Response

<ResponseField name="success" type="boolean">
  Indicates whether the request was successful.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data properties">
    <ResponseField name="withdrawn" type="boolean">
      `true` when the invitation was successfully withdrawn.
    </ResponseField>
    <ResponseField name="invitation_id" type="string">
      The ID of the withdrawn invitation (echo of the input).
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="metadata" type="object">
  <Expandable title="metadata properties">
    <ResponseField name="action" type="string">The action that was performed (`"withdraw"`).</ResponseField>
    <ResponseField name="account_id" type="string">The account ID used for the request.</ResponseField>
    <ResponseField name="credits_consumed" type="integer">Number of credits consumed (1).</ResponseField>
    <ResponseField name="timestamp" type="string">ISO 8601 timestamp of the response.</ResponseField>
  </Expandable>
</ResponseField>
