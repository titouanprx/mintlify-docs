---
title: "Source Hiring Companies"
api: "POST /v1/data/companies/hiring"
description: "Search for companies actively hiring by scraping job boards (Indeed and LinkedIn)"
---

## Overview

This endpoint retrieves companies that are actively hiring with their job listings, company details, and enriched data.

**At least one** of `job_title`, `job_description`, or `location` is required.

<Note>
  **Pricing:** Total Cost = Base + Description + Enrichment
  
  - **Base**: 1 credit = 10 companies
  - **Description** (if `fetch_description: true`): +1 credit per 10 companies
  - **Enrichment** (if `enrich_company: true`): +1 credit per company
  
  **Examples for 10 companies:**
  - Base only: **1 credit**
  - Base + Description: **2 credits**
  - Base + Enrichment: **11 credits**
  - Full (Base + Description + Enrichment): **12 credits**
</Note>

## Body Parameters

<ParamField body="company_names" type="array">
  Target specific companies by name (e.g., `["Google", "Amazon"]`)
  
  <Note>
    When this parameter is provided, only **LinkedIn** will be scraped as the job board, regardless of the `site_names` parameter.
  </Note>
</ParamField>

<ParamField body="job_title" type="array">
  Keywords for job title (e.g., `["developer", "engineer"]`)
</ParamField>

<ParamField body="job_title_include_all" type="boolean" default="false">
  If `true`: all keywords required (AND logic). If `false`: at least one keyword (OR logic)
</ParamField>

<ParamField body="job_title_excluded" type="array">
  Keywords to exclude from job title (e.g., `["senior", "manager"]`)
</ParamField>


<ParamField body="location" type="string">
  Location (city, region). Example: `"Paris, France"` or `"San Francisco, CA"`
</ParamField>


<ParamField body="is_remote" type="boolean" default="false">
  Filter only remote jobs
</ParamField>

<ParamField body="country" type="string" default="france">
  Country for Indeed. Values: `france`, `usa`, `uk`, `germany`, `spain`, `italy`, `canada`
</ParamField>

<ParamField body="job_type" type="string">
  Type of contract. Values: `fulltime`, `parttime`, `internship`, `contract`
</ParamField>

<ParamField body="hours_old" type="integer">
  Jobs posted within the last X hours (e.g., `72` for 3 days)
</ParamField>

<ParamField body="site_names" type="array" default='["indeed", "linkedin"]'>
  Job boards to scrape. Values: `linkedin`, `indeed`
</ParamField>

<ParamField body="total_results" type="integer" default="100">
  Number of companies to retrieve
</ParamField>

<ParamField body="fetch_description" type="boolean" default="true">
  Fetch complete job descriptions. **Cost: +1 credit per 10 companies**
</ParamField>

<ParamField body="enrich_company" type="boolean" default="false">
  Enrich with LinkedIn data (tagline, followers, HQ, locations). **Cost: +1 credit per company**
</ParamField>

<ParamField body="start_page" type="integer" default="1">
  Start page for pagination
</ParamField>

<ParamField body="end_page" type="integer">
  End page for pagination
</ParamField>

<RequestExample>

```bash Basic Search (1 credit)
curl -X POST "https://api.linkupapi.com/api/v2/web-crawler/source-hiring-company" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "job_title": ["developer"],
    "location": "Paris, France",
    "total_results": 10,
    "fetch_description": false,
    "enrich_company": false
  }'
```

```bash With Enrichment (12 credits)
curl -X POST "https://api.linkupapi.com/api/v2/web-crawler/source-hiring-company" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "job_title": ["frontend developer"],
    "job_title_excluded": ["senior", "lead"],
    "location": "Berlin, Germany",
    "country": "germany",
    "is_remote": true,
    "job_type": "fulltime",
    "hours_old": 168,
    "site_names": ["linkedin", "indeed"],
    "total_results": 10,
    "fetch_description": true,
    "enrich_company": true
  }'
```

```bash Target Specific Companies (LinkedIn only)
curl -X POST "https://api.linkupapi.com/api/v2/web-crawler/source-hiring-company" \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "company_names": ["Google", "Amazon", "Meta"],
    "job_title": ["software engineer"],
    "location": "San Francisco, CA",
    "is_remote": true,
    "total_results": 50,
    "fetch_description": true,
    "enrich_company": true
  }'
```
