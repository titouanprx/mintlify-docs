---
title: 'Extract Profile Posts'
api: 'POST /v1/data/signals/profile/posts'
description: 'Extract posts from a LinkedIn profile'
---

Extract posts from a LinkedIn profile with various filtering and sorting options.

<Note>
  **Credit Usage:** 1 credit per request
</Note>

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="profile_url" type="string" required>
  URL of the LinkedIn profile to extract posts from
</ParamField>

<ParamField body="proxy_country" type="string" default="FR">
  Country code for proxy selection. Available: (US, UK, FR)
</ParamField>

<ParamField body="total_results" type="integer" default="10">
  Number of posts to retrieve (used when not in pagination mode)
</ParamField>

<ParamField body="post_type" type="string" default="ALL">
  Type of posts to filter. Available: (ALL, ARTICLE, VIDEO, PHOTO, DOCUMENT)
</ParamField>

<ParamField body="sort_by" type="string" default="DATE_POSTED">
  Sort order for posts. Available: (RELEVANCE, DATE_POSTED)
</ParamField>

<ParamField body="keyword" type="string">
  Keyword to filter posts by content
</ParamField>

<ParamField body="post_date" type="string">
  Date filter for posts. Available: (PAST_24_HOURS, PAST_WEEK, PAST_MONTH)
</ParamField>

<ParamField body="use_pagination" type="boolean" default="false">
  Use pagination mode instead of total results mode
</ParamField>

<ParamField body="start_page" type="integer">
  Starting page number for pagination mode (required if use_pagination is true)
</ParamField>

<ParamField body="end_page" type="integer">
  Ending page number for pagination mode (optional if use_pagination is true)
</ParamField>

### Notes

- The API supports LinkedIn profile URLs in the format `linkedin.com/in/username`
- Results can be filtered by post type, sorted by relevance or date, and filtered by keywords
- Date filters allow you to focus on recent posts
- **Pagination Mode vs. Total Results Mode**:
  - **Total Results Mode**: Use the `total_results` parameter to specify how many posts you want (default)
  - **Pagination Mode**: Set `use_pagination` to true and use `start_page` and `end_page` parameters
- Media URLs are provided for posts with images, videos, or documents
- Article posts include additional metadata like article URL and title
- Rate limits apply to profile post extraction requests
