---
title: 'Extract Post Comments'
api: 'POST /v1/data/signals/posts/comments'
description: 'Extract comments from a specific LinkedIn post'
---

Extract detailed comments from a LinkedIn post, including information about who commented and the comment content.

<Note>
  **Credit Usage:** 1 credit per 10 results (or fraction thereof). For example:
  - 1-10 results = 1 credit
  - 11-20 results = 2 credits
  - 300 results = 30 credits
</Note>


### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="post_url" type="string" required>
  URL of the LinkedIn post to extract comments from
</ParamField>

<ParamField body="proxy_country" type="string" default="FR">
  Country code for proxy selection. Available: (US, UK, FR, DE, NL, IT, IL, CA, BR, ES, IN)
</ParamField>

<ParamField body="total_results" type="integer" default="10">
  Number of comments to retrieve (used when not in pagination mode)
</ParamField>

<ParamField body="use_pagination" type="boolean" default="false">
  Use pagination mode instead of total results mode
</ParamField>

<ParamField body="start_page" type="integer">
  Starting page number for pagination mode (required if use_pagination is true)
</ParamField>

<ParamField body="end_page" type="integer">
  Ending page number for pagination mode (optional if use_pagination is true)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Request status (success/error)
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Properties">
    <ResponseField name="post_url" type="string">
      URL of the analyzed post
    </ResponseField>

    <ResponseField name="total_results" type="integer">
      Number of comments returned in this response
    </ResponseField>

    <ResponseField name="total_available" type="integer">
      Total number of comments available on the post
    </ResponseField>

    <ResponseField name="comments" type="array">
      <Expandable title="Comment Object">
        <ResponseField name="comment_urn" type="string">
          LinkedIn URN identifier for the comment
        </ResponseField>

        <ResponseField name="tracking_id" type="string">
          Internal tracking identifier for the comment
        </ResponseField>

        <ResponseField name="comment_text" type="string">
          The comment content
        </ResponseField>

        <ResponseField name="created_time" type="integer">
          Unix timestamp when the comment was posted
        </ResponseField>

        <ResponseField name="commenter" type="object">
          <Expandable title="Commenter Properties">
            <ResponseField name="name" type="string">
              Full name of the comment author
            </ResponseField>

            <ResponseField name="profile_urn" type="string">
              LinkedIn URN identifier for the commenter's profile
            </ResponseField>

            <ResponseField name="linkedin_url" type="string">
              LinkedIn profile URL of the commenter
            </ResponseField>

            <ResponseField name="occupation" type="string">
              Professional headline or occupation
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

### Notes

- The API supports LinkedIn post URLs in various formats
- Results are paginated with configurable page ranges when using pagination mode
- The `created_time` field is provided as a Unix timestamp
- Each comment includes a unique `comment_urn` and `tracking_id` for identification
- The `commenter` object contains detailed information about the comment author
- **Pagination Mode vs. Total Results Mode**:
  - **Total Results Mode**: Use the `total_results` parameter to specify how many comments you want (default)
  - **Pagination Mode**: Set `use_pagination` to true and use `start_page` and `end_page` parameters
- Rate limits apply to comment extraction requests
- The `total_available` field shows the total number of comments available on the post
