---
title: 'Email Reverse Lookup'
api: 'POST /v1/data/mail/reverse'
description: 'Find LinkedIn profile information from an email address'
---

Find and extract LinkedIn profile information using an email address. This endpoint parses the email, extracts company information, and searches for the corresponding LinkedIn profile.

<Note>
  This endpoint costs **1 credit** per request.
</Note>

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="email" type="string" required>
  Email address to reverse lookup (e.g., yoan@limova.ai)
</ParamField>

### Response

<ResponseField name="status" type="string">
  Request status (success/error)
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Properties">

            <ResponseField name="start_date" type="string">

### Notes
- The endpoint attempts to parse names and company information from the email address
- If no LinkedIn profile is found, `person_found` will be `null`
- Parsing confidence indicates how reliable the name extraction was
- Profile data depends on LinkedIn privacy settings
- Contact information may be empty based on profile visibility settings
