---
title: 'Profile Enrichment'
api: 'POST /v1/data/profil/enrich'
description: 'Find and extract comprehensive profile information using name and company'
---

Automatically find a person's profile using their name and company, then extract complete profile information in a single request. This endpoint combines profile search and data extraction in one seamless process.

<Note>
  This endpoint costs **1 credit** per request.
</Note>

### Header Parameters

<ParamField header="x-api-key" type="string" required>
  Your API key
</ParamField>

### Body Parameters

<ParamField body="first_name" type="string" required>
  Person's first name
</ParamField>

<ParamField body="last_name" type="string" required>
  Person's last name
</ParamField>

<ParamField body="company_name" type="string" required>
  Current company name where the person works
</ParamField>

### Response

<ResponseField name="status" type="string">
  Request status (success/error)
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Properties">
    <ResponseField name="person_searched" type="object">
      <Expandable title="Search Criteria">

        <ResponseField name="industry" type="string">
          Industry sector

### Notes
- The endpoint tries multiple search methods to maximize success rate
- If no profile is found, `linkedin_profile` and `full_profile_data` will be `null`
- Contact information (email, phone) may be empty
